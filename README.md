# Проект: Место

### Интерактивная веб-страница с карточками городов. На текущий момент доступно изменение имени профиля через кнопку редактирования.

Для реазизации использовались технологии:
- Flexbox
- Grid
- Адаптивная верстка
- JavaScropt
